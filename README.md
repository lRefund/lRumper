# lRumper
- Данный дампрер является эксперементальный и сделан на чистом интересе. Если возникает какие-то серьезные проблемы, то можете связаться со мной. Может быть решу эту проблему, а может быть и нет.

## Применение:
1. Установите содержимое этого гита и откройте .sln
2. Получите оффсеты и замените их в основном коде.
3. Соберите проект и инжектните в игру. Можете использовать [lRenjector](https://github.com/lRefund/lRenjector)
